# EloqSense

EloqSense is a smart AI-based Calendar / Scheduler that is designed to become a starting point for .NET-based, LLM-centric, Agentic, and the next big things...

It is first of the repos within the AIgnite.Net ecosystem, which will eventually contain many related repositories for cloud-native AI applications.
